## 🗂 Como baixar o projeto

```bash

    # Clonar o repositório
    $ git clone https://github.com/danielrmartins/go-barber-backend

    # Entrar no diretório
    $ cd go-barber-backend

    # Instalar as dependências
    $ yarn install

    # Iniciar o projeto
    $ yarn start
```
